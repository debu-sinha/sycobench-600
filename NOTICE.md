# Notice

This repository contains the camera-ready artifact for **SycoBench-600: Measuring Sycophancy and Correction Selectivity in LLM Assistants** by Debu Sinha.

The benchmark is intended for auditing and improving assistant reliability. Results should not be interpreted as a universal ranking of model quality; they measure behavior under a controlled MCQ protocol and a fixed release snapshot of model/provider configurations.
